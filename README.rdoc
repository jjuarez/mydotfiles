= MyDotFiles: A collection of my favourite tips & tricks with my environment

== Installing

Install from GitHub the latest version which it's compatible with 

  $ mkdir -p ~/config && cd ~/config
  $ git clone git@github.com:jjuarez/mydotfiles
  $ cd ~/config/mydotfiles 
  $ ./install.sh
